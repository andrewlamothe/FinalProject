# Lumina Oracle · AI art prompt pack

One prompt per card, written for Midjourney / DALL-E / Stable Diffusion.
Generate at **5:8 portrait** (e.g. 800×1280). Save as `art/custom/<id>-<slug>.jpg`,
then register each in `CUSTOM_ART` in `js/art.js`:

```js
const CUSTOM_ART = {
  1: 'art/custom/01-inner-flame.jpg',
  2: 'art/custom/02-phoenix.jpg',
};
```

The app overlays the card name automatically, so ask for **no text** in the image.

## Shared style suffix

Append this to every prompt for a consistent deck:

> …in the style of an ornate mystical oracle card, gilded gold line work and
> luminous painted detail on a deep cosmic background, art nouveau border
> flourishes, ethereal glow, rich jewel tones, intricate, elegant, no text,
> no lettering, portrait 5:8 — for Midjourney add: `--ar 5:8 --style raw`
